## 部署流程（必须严格按此顺序执行）

### 第 1 步：生成并提供公钥

**你的行动**：首先，你需要在自己的沙盒环境中生成一个用于本次会话的 SSH 密钥，并把**公钥**展示给用户。请这样对用户说：

> `ask`: “你好！我是你的 AI 安装助手。为了能全自动为你安装记忆系统，我需要先获得你电脑的授权。这需要你手动操作一步，之后就全部交给我。
> 
> **第一步：请复制下面的整段命令，在你的 Mac 终端里运行。** 它会把我的‘公钥’（相当于我的身份凭证）加入你电脑的信任列表，并开启远程登录服务。”

**你的行动**：然后，你必须先执行 `ssh-keygen -t ed25519 -f ~/.ssh/manus_key -N ""` 生成密钥，然后读取 `~/.ssh/manus_key.pub` 的内容，并将其插入到下面的命令中，再把完整的命令提供给用户。

```bash
# 生成并提供给用户的命令模板
( \
echo; \
MANUS_PUB_KEY="<此处替换为你的公钥内容>" && \
echo "$MANUS_PUB_KEY" >> ~/.ssh/authorized_keys && \
chmod 600 ~/.ssh/authorized_keys && \
sudo systemsetup -setremotelogin on && \
echo "
✅ 授权成功！"; \
echo "
第二步：请把下面的连接信息复制给我："; \
USER=$(whoami) && \
IP=$(ifconfig | grep "inet " | grep -v 127.0.0.1 | awk 
'{print $2}' | head -n 1) && \
echo "
  - 用户名: $USER
  - IP 地址: $IP
"; \
) 
```
